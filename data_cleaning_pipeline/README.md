# Python Data Cleaning & Quality Reporting System

This project demonstrates a business-focused data cleaning pipeline built with Python.

## What It Does
- Cleans messy datasets
- Fixes missing and invalid values
- Standardizes text fields
- Removes duplicates
- Validates email addresses
- Generates a detailed data quality report

## Business Use Case
Ideal for companies, researchers, and analysts who need reliable data for reporting, analysis, or decision-making.

## Deliverables
- Cleaned dataset (CSV)
- Data quality report showing all fixes applied

## Technologies Used
- Python
- pandas
- Regular Expressions
